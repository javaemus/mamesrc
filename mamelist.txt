This is a list of all ROM sets currently supported by MAME.  Additional fields are:

rom -- the name of the ROM set as recognized by MAME
drivers -- the name of the C file from the drivers directory that runs the ROM set
machine -- the name of the C file from the machine directory that runs the ROM set
sndhrdw -- the name of the C file from the sndhrdw directory that runs the ROM set
vidhrdw -- the name of the C file from the vidhrdw directory that runs the ROM set
sound -- the method used to generate sound for this ROM set
cpus -- the number and type of CPUs emulated for this ROM set
name -- the full name of the emulated system

This list was compiled by Aaron Giles. If you have any updates or new information, please
email it to me at agiles@sirius.com
